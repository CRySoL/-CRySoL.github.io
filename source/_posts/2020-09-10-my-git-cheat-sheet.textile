h4. Delete all remote branches


<pre class="console">
$ git branch -a | grep -v master | awk -F'/' '{print $3}' | xargs git push origin --delete
</pre>


h4. Sync remote and local branches

<pre class="console">
$ git fetch --prune
</pre>

Option 'prune' removes local branches absent in remote.
