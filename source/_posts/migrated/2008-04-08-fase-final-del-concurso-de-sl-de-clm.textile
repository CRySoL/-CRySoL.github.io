Desde la UCLM y el Centro de Excelencia de Software Libre de Castilla-La Mancha (CESLCAM) se está organizando la Fase Final del I Concurso Universitario de Software Libre de Castilla-La Mancha que tendrá lugar el próximo 17 de abril en la Escuela Superior de Informática de Ciudad Real.

Junto a la entrega de premios, se van a realizar charlas y diversos talleres relacionados con el software libre.

*9:00 – 09:30* :: Bienvenida
*09:30 – 10:00* :: Inauguración Exposición "Por Tierras de Molinux"
*10:00 – 11:00* :: Presentación 3 Proyectos Finalistas
*11:00 – 11:30* :: Pausa Café
*11:30 – 12:30* :: Presentación 3 Proyectos Finalistas
*12:30 – 14:00* :: Presentación PCYTA / Taller "Gestor contenidos web Joomla" (Ontic Soluciones)
*14:00 – 16:00* :: Descanso
*16:00 – 17:00* :: Taller "Ataque Web 2.0" (Area Project Solutions)
*17:00 – 17:45* :: Taller "Introducción a la programación de videojuegos" (Tobías Díaz)
*18:00 – 19:00* :: Entrega de Premios

Los alumnos de la UCLM que asistan pueden obtener además créditos de libre configuración
