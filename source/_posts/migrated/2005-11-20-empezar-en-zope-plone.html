OBSOLETA: Mira Debian Plone en 5 minutos
Trataremos de ver los primeros pasos a la hora de tratar con Zope y Plone. El artículo quizá esté un poco viejo, pero puede ayudar.

Usuario de emergencia

Lo primero es instalar ZOPE y los modulos que mas te gusten :) (usa apt-get) Para crear el usuario de emergencia, necesitaremos ser root. Primero vamos al directorio /var/lib/instance/default, donde ejecutamos la aplicacion zope-zpasswd y pasamos como nombre de fichero "access":
{% highlight console %} localhost:/var/lib/instance/default# zope-zpasswd access {% endhighlight %}
Nos pedirá un nombre de usuario INICIAL (no administrador!) y su password, a las siguientes dos cuestiones pulsamos intro y ya esta... Es necesario reiniciar Zope!! Para reiniciar Zope, existen dos formas (ambas como root):
{% highlight console %} # /etc/init.d/zope restart {% endhighlight %}
o bien
{% highlight console %} # zopectl restart {% endhighlight %}

Entrar en la web de administración

Desde el navegador, la url es: http://localhost:9673/manage . Nos pedira el nombre y password del usuario inicial, una vez dentro podremos crear el usuario administrador y el resto de usuarios... y ya está! ;)
