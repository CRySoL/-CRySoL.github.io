LG L204-WT en Debian

Cómo sacar todo el partido a este monitor en XOrg.

Ingredientes

  • Un monitor LG L204-WT
  • Debian GNU/Linux

A mi me daban dos

Este es el fichero xorg.conf con el driver privativo de nvidia para dos monitores, es decir, salidas VGA + DVI de una tarjeta dual head. Con esto tenemos un escritorio de 3360 × 1050, es decir, la resolución mínima recomendada para “vim” :-).
