Buenas... los debianitas habrán notado como hace algún tiempo que el maravilloso y útil "Menú Debian" ha desaparecido, no se porqué... pero si cómo hacerlo volver.... :D

Menú a la carta

¿El problema es que queremos una opción en el menú que ya no tenemos? ¿Queremos nuestro menú a la carta? pues instalemos alacarte (qué! bromaza :P):
{% highlight console %} # apt-get install alacarte {% endhighlight %}
O si usamos aptitude:
{% highlight console %} # aptitude install alacarte --please-dont-destroy-my-system {% endhighlight %}
Si ahora ejecutamos alacarte (como usuario también vale, lo activaremos para nuestra cuenta) vemos que ¡¡el "Menú Debian" está ahí... pero desactivado!!... pues lo activamos y ya está...
