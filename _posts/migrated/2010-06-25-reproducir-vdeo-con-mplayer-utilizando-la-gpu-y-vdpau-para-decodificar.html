Con esta receta dejaremos la carga de decodificar el vídeo a la GPU, con lo que notaremos una gran mejoría en la fluidez de reproducción del vídeo y una descarga de trabajo en la CPU.

Compatibilidad

Esta receta está destinada a tarjetas NVIDIA con soporte para vdpau, en esta tabla podrás comprobar si tu tarjeta tiene soporte.

Pasos a seguir:

  1. Instalar driver de nvidia: Bien explicado en esta receta
  2. Instalar driver nvidia-vdpau:
    {% highlight console %} # apt-get install nvidia-vdpau-driver {% endhighlight %}
  3. Instalar mplayer:
    {% highlight console %} # apt-get install mplayer {% endhighlight %}
  4. Reproducir vídeo:
    {% highlight console %} $ mplayer -vo vdpau -vc ffh264vdpau video_a_reproducir {% endhighlight %}

front-end

Si quieres utilizar un front-end, yo recomiendo smplayer. Para utilizar vdpau con smaplayer accede a configuración (botón del destornillador) y en esa ventana: General > Video > Driver de salida (estará la opción xv) substituye por vdpau y aplicar.
Aclaración

vdpau no es CUDA; se puede comprobar en la tabla enlazada al inicio de la receta que tarjetas gráficas sin tecnología CUDA tienen soporte para utilizar la biblioteca VDPAU (Video Decode and Presentation API for Unix).
