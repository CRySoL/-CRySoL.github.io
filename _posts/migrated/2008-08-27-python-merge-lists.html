--- migrated: node/993 layout: post title: ! 'Python: merge lists' created: 1219870243 author: david_villa category: scriptillo tags: - Python --- A simple utility function to merge several lists in a single one, using the functional programming tools:
{% highlight python %} def merge(*input): return reduce(list.__add__, input, list()) {% endhighlight %}
Example:
{% highlight python %} >>> a = [0, 1, 2] >>> b = [2, 3, 4] >>> c = [4, 5, 6] >>> merge(a, b, c) [0, 1, 2, 2, 3, 4, 4, 5, 6] {% endhighlight %}
The same but removing duplicates:
{% highlight python %} def merge_uniq(*input): return list(reduce(set.union, input, set())) {% endhighlight %}
The same but removing duplicates later:
{% highlight python %} def merge_uniq(*input): return list(set(merge(input))) {% endhighlight %}
