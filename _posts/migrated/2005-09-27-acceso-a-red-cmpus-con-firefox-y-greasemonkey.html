--- migrated: node/14 layout: post title: Acceso a Red C@mpus con Firefox y Greasemonkey created: 1127835076 author: Geronimo comments: true category: recipe --- Todos los alumnos y profesores de la UCLM que usan otro navegador web que no sea Internet Explorer saben lo "peculiar" que es la web de RedC@mpus. Esta receta explica cómo arreglar este incomprensible error. Para permitir a los usuarios de Mozilla y Firefox acceder a esa web, he escrito un script para la extensión Greasemonkey de Firefox/Mozilla/etc. que permite usar Red-C@mpus desde estos navegadores. Básicamente lo que hace es sustituir el menú "desplegable" que aparecía al situar el cursor encima de un enlace en la página principal de Red-C@mpus por un menú horizontal de navegación.

¿Qué es Greasemonkey?

Según la propia web de Greasemonkey:
Greasemonkey is a Firefox extension which lets you to add bits of DHTML ("user scripts") to any web page to change its behavior. In much the same way that user CSS lets you take control of a web page's style, user scripts let you easily control any aspect of a web page's design or interaction.

Instalar Greasemonkey

En la página de Complementos para Firefox, busca Greasemonkey y pincha en el enlace Añadir a Firefox y reinicia a continuación el navegador.

Instalar el Script

Userscripts.org es un punto de encuentro para la comunidad de usuarios de Greasemonkey. En este sitio podrás encontrar miles de scripts, y entre ellos Free-C@mpus. Visita el enlace anterior y pulsa en el enlace Install, y cuando aparezca una ventana emergente mostrando información acerca del script, pulsa en Aceptar para confirmar la instalación.
