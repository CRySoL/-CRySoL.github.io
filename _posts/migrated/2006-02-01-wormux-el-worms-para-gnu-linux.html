Se trata del Worms de toda la vida pero para GNU/Linux y libre. Tiene muy buena pinta, aunque carece de ciertos aspectos. Las animaciones no están muy curradas y hay ciertos movimientos que no están permitidos (subirse a un jugador). Por lo demás, decir que es un fantástico juego y que la gente que lo ha desarrollado se lo ha currado. Realmente las carencias que tiene son pequeños matices; sin ninguna importancia. El juego en sí está muy logrado. ¡Enhorabuena! Para instalarlo, teneis que agregar en /etc/apt/source.list: 
deb http://download.gna.org/wormux/debs sarge/  # Debian stable ("Sarge")
deb http://download.gna.org/wormux/debs sid/    # Debian unstable ("Sid")
deb http://download.gna.org/wormux/debs breezy/ # Ubuntu Breezy
deb http://download.gna.org/wormux/debs dapper/ # Ubuntu Dapper
¡Agregar sólo la que utiliceis vosotros!. También hay para otras plataformas. Una vez hecho esto, debeis hacer un apt-get update y, depués, apt-get install wormux. Espero que os guste. Un saludo!
