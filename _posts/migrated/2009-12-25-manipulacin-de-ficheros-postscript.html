Convertir un documento a formato de varias páginas por hoja

Utilizando el programa psnup del paquete psutils:
{% highlight console %} $ psnup -2 original.ps salida.ps {% endhighlight %}

Seleccionar algunas páginas

{% highlight console %} $ psselect -p 3,7,12-23,34 original.ps resultado.ps {% endhighlight %}
