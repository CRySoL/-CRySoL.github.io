¿Qué es Eagle mode?

Según la página de Eagle Mode, se trata de una solución avanzada para un estilo futurístico de comunicación hombre-máquina. Bueno... si habéis visto el vídeo supongo que opinaréis como yo en que por ahora es más una forma de quedarse con el personal que de usar un PC... pero bueno... es muy bonito... Cuando uno ve el vídeo, algunas de las preguntas que le surgen son las siguientes:
  • ¿Pero que características tiene que tener mi PC para mover esto? ¡Es realmente increíble lo ligero que resulta!
  • ¿Cuánto ocupa esta monstruosidad? La friolera de 3.2M comprimido y 19M una vez compilado.
  • ¿Realmente es fácil de manejar? Pues no... con el shift suavizamos los movimientos... pero aún así cuesta pillarle el truco
  • ¿Y para compilarlo? Tener algunos paquetes -dev y ejecutar un script...
  • ¿Hay que pasarse horas configurando todo? Compilar y listo para usar!!
  • ¿La licencia es libre? Nada menos que la GPL v3.
Como veréis... algo tan sencillo merece la pena probar!, así que no he podido resistirme a escribir la receta...

Descargar

En la sección de descargas de su página oficial podemos encontrar un LiveCD y el código fuente de la aplicación.

Compilar

Antes tendremos que asegurarnos de tener instalados algunos paquetillos (con todos estos tendremos todas las funcionalidades... pero no es obligatorio tenerlos todos):
{% highlight console %} $ sudo aptitude install perl g++ libx11-dev xterm gs libjpeg62-dev libpng12-dev libtiff4-dev libxine-dev p7zip-full binutils arc arj bzip2 gzip lha lzop tar unrar zip unzip zoo abiword tetex-bin transfig netpbm htmldoc genisoimage gs-common povray rpm librsvg2-bin libwmf-bin {% endhighlight %}
Ya solo nos queda descomprimir los sources y compilar:
{% highlight console %} $ unp eaglemode-0.75.0.tar.bz2 $ cd eaglemode-0.75.0 $ ./make.pl build {% endhighlight %}
...si todo va bien... ¡ya lo tenemos listo!

Pruebas

Después de compilar sólo nos queda probarlo...
{% highlight console %} $ ./eaglemode.sh {% endhighlight %}
...o mejo aún...
{% highlight console %} $ ./eaglemode.sh -fullscreen {% endhighlight %}

Instalación

Si queremos instalarlo en el sistema:
{% highlight console %} # ./make.pl install {% endhighlight %}
...o si lo queremos hacer como mandan los cánones...
{% highlight console %} # ./make.pl install dir=/opt/eaglemode {% endhighlight %}
Y ya está! ultrasimple!
