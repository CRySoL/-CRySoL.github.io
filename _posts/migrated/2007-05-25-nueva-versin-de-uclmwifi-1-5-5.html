Me es grato comunicaros que "uclmwifi" sigue avanzando. La gente que ya se instaló la versión 1.2 habéis "sufrido" las distintas actualizaciones y cambios que ha sufrido uclmwifi. Si aún no lo has probado añade el siguiente repositorio: 
deb http://babel.esi.uclm.es/gnesis unstable main
deb-src http://babel.esi.uclm.es/gnesis unstable main
Y con lo siguiente, instala uclmwifi:
{% highlight console %} # apt-get update # apt-get install uclmwifi {% endhighlight %}
En esta versión (1.5.5), uclmwifi ya es un paquete en condiciones (gracias a la ayuda inestimable de Paco y de David) y presenta las siguientes novedades:
  • El archivo "vpn-mppe-utils" contiene funciones para crear un túnel mppe para conexiónes wireless o cableadas. Sólo debe emplearse si piensas crear tu propia conexión. Los scripts "uclmwifi" y "uclmvpn" pueden servirte como ejemplo.
  • "uclmwifi" llama a las funciones de "vpn-mppe-utils" en el orden apropiado y con los parámetros correctos para realizar la conexión a la VPN vía wireless.
  • "uclmvpn" hace lo mismo que "uclmwifi" pero para conexiones externas (desde fuera de la Uni). Es necesario tener red para que se ejecute correctamente. ¡OJO! Los alumnos no tienen permiso para conexiones externas (al menos yo :-)). Falta comprobar si las cuentas de profesores funcionan.
  • Todo viene bien explicadito con páginas de man ;-).
  • Para reconfigurar el usuario y contraseña de la conexión basta con "dpkg-reconfigure uclmwifi".
  • Se han arreglado diversos bugs detectados en distintas situaciones de conexión (problemas con DHCP, asociación con puntos de acceso...). Comprueba el Changelog para más información.
uclmwifi forma parte del desarrollo de un applet para GNOME llamado vpnapplet. El proyecto está alojado en GNA y puedes descargarlo utilizando SVN con:
{% highlight console %} svn co svn://svn.gna.org/svn/vpnapplet vpnapplet {% endhighlight %}
Gracias a todos por usar el script y por vuestras sugerencias. ¡Un saludo! Pd: ¡Mirad! uclmwifi llega a ubuntu :-)
