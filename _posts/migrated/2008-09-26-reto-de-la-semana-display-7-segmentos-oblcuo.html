--- migrated: node/1014 layout: post title: 'Reto de la semana: "display 7 segmentos oblícuo"' created: 1222425384 author: david_villa comments: true category: challenge render_with_liquid: false --- Pues nada, se trata de escribir un programilla en el lenguaje que quieras, que tome por línea de comandos un número de cualquier tamaño (menor que un googolplex) y lo represente como en el siguiente ejemplo:
{% highlight console %} $ pinta_num 1234567890 \ /\ /\ \ / / /\ /\ /\ /\ \ / /\\/\\/\\/\ \\/\\/\\ \ \/ / / \/ \/ / \/ {% endhighlight %}
Debe evitarse el uso de secuencias de escape del terminal tipo "gotoxy". Si se redirige la salida a un fichero, debe escribir lo mismo que en consola. Se valora la legibilidad y brevedad del código por encima de cualquier otra consideración. *[EDITADO]* Alternativa de representación, que no tengo muy claro si es más fácil o más difícil:
{% highlight console %} $ pinta_num 1234567890 /\ \ \ /\ \/ \/\ /\ / \/\ /\ \/ \ / \/\ / \/ \/\ \ / \/\ /\ /\ /\ / / \ \/ \ {% endhighlight %}
