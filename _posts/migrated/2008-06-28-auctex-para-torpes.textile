h2. Modo PDF-Tex

<!--break-->


Se puede conseguir añadiendo lo siguiente a tu <tt>.emacs</tt>


<div>
{% highlight text %}
(setq TeX-PDF-mode t)
{% endhighlight %}
</div>


h2. Combinaciones

C-c C-c: Compilar, ver el resultado
C-c C-p C-b: Preview en buffer
C-c ]: Cerrar entorno

C-c C-s: Secciones
C-c C-e: Entornos


h2. Usar otro visor para los PDF


<div>
{% highlight text %}
M-x customize-group
TeX-command
Tex Output View Style 
Extension ^pdf$
Command: evince %o %(outpage)
{% endhighlight %}
</div>
