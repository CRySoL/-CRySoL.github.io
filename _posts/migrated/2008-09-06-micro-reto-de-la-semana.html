{% highlight c %} printf("hola" + 2); {% endhighlight %}
C++:
{% highlight cpp %} cout << "hola" + 2; {% endhighlight %}
Java:
{% highlight java %} System.out.print("hola" + 2); {% endhighlight %}
Python, Perl, Ruby (mismo programa, distintos resultados)
{% highlight python %} print "hola" + 2 {% endhighlight %}
c-shell:
{% highlight text %} echo "hola" + 2 {% endhighlight %}
C#:
{% highlight text %} Console.WriteLine("hola" + 2); {% endhighlight %}
LUA:
{% highlight text %} print ("hola" + 2) {% endhighlight %}
Lo de "qué imprime" se puede considerar una pregunta retórica ya que con escribir el programa correspondiente ya tienes la solución. La interesante es sale lo que esperabas y sobre todo el porqué...
