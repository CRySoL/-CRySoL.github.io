Preparando el núcleo

Si tienes una Debian y tu núcleo es el 2.6.14-2 puedes bajarte el mppe-linux-modules-2.6.14-2.deb para tener soporte MPPE en tu núcleo. ¡Ojo!, esto está probado para la versión 2.6.14-2… si lo haces con otro núcleo MUY probablemente no funcione. Para instalarlo utiliza:
{% highlight console %} # dpkg -i nombre_del_paquete.deb {% endhighlight %}
Si hay alguna dependencia incumplida haz:
{% highlight console %} # apt-get -f install {% endhighlight %}

Si utilizas Ubuntu...

La gente que utiliceis núcleos ya parcheados con MPPE (como Ubuntu y demás…) no hace falta que hagais lo anterior.

Bajando e instalando el script

Cuando tu núcleo tenga soporte MPPE es hora de bajarte el UclmWiFi y se instala igual que el anterior. Si tienes problemas de dependencias haz lo que te he explicado anteriormente.

Ejecutando el script

Una vez configurado al hacer:
{% highlight console %} # activawifi.sh {% endhighlight %}
Puede ser que salga mensajes de error en la consola. No te preocupes. Son comprobaciones que se hacen y que no repercuten en la ejecución. Sólo tienes que esperar a un mensaje de éxito o de fallo del programa. Cuando quieras finalizar la conexión con la WiFi sólo has de hacer:
{% highlight console %} # poff {% endhighlight %}
