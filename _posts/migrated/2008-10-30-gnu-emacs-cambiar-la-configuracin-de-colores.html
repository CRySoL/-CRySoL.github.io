Instalación

En Debian/Ubuntu:
{% highlight console %} apt-get install emacs-goodies-el {% endhighlight %}
Para distribuciones no basadas en Debian, descarga el archivo color-theme.el de aquí y colócalo en tu LoadPath.

Configuración y uso

Introducir el siguiente comando en el minibuffer de emacs:
{% highlight text %} META-x color-theme-select {% endhighlight %}
Inmediatamente se abre un buffer donde puedes seleccionar y probar cada uno de los temas existententes. Si lo prefieres puedes editar el archivo ~/.emacs e introducir la siguiente línea:
{% highlight text %} (require 'color-theme) {% endhighlight %}
A partir de ahora con META-x color-theme-TAB puedes elegir el tema que quieras directamente desde el minibuffer. Si quieres tener un tema fijo puedes introducir también la siguiente línea en ~/.emacs (aunque a mí de esta forma no me carga muy bien los temas):
{% highlight text %} (color-theme-initialize) (color-theme-nombre-del-tema) {% endhighlight %}
